# ScanLogin
## 简介
ScanLogin是thymelte系统的android端代码。
## 已实现的功能
- 登录退出
- 扫码
- Android6.0以上设备的权限检查

## 使用流程
必须修改`AppConfig`配置的`baseUrl`变量的ip地址，为你本机的真实ip地址。
